# 4DOF Robotic Arm - SolidWorks to MATLAB Simscape

## 📌 Project Overview
This project presents the design, simulation, and control of a 4-DOF robotic manipulator. The system is modeled in SolidWorks and exported to MATLAB using Simscape Multibody Link for dynamic simulation and control.

## 🛠️ Features
- 3D Mechanical Design using SolidWorks
- Export to MATLAB via Simscape Multibody Link
- Dynamic simulation using Simscape Multibody
- Manual gain tuning for stability
- Forward kinematics visualization based on joint angles
- Interactive control of joints

## 🧩 System Architecture
The robotic arm consists of:
- Base
- Link 1
- Link 2
- Link 3
- End-effector (Gripper with two fingers)

Total DOF: **4 Degrees of Freedom**

## ⚙️ Requirements
Before running the project, make sure you have:

- MATLAB (R2023 or lower recommended)
- Simscape
- Simscape Multibody
- Simscape Multibody Link Plugin (SolidWorks)
